## Executable and Linkable Format (ELF)

ELF adalah sebuah file format yang digunakan pada sistem Unix untuk
file executable, object code, shared libraries dan juga core dumps.

Ketika kamu menjalankan sebuah program di komputer Linux mu, kemungkinan
besar kamu telah mengeksekusi sebuah file ELF.

### Layout File ELF

Sebuah file ELF terdiri dari berbagai sections, diantaranya sebagai berikut:

`.text`: tempat instruksi-instruksi program disimpan  
`.data`: tempat variabel global yang terinisialisasi  
`.bss`: tempat variabel global yang tidak terinisialisasi  
`.shrtrtab`: tabel string yang dipakai di program  

Setiap section tersebut mempunyai address yang menunjukkan dimana section
tersebut berada di memori.

:::info
Namun, dalam executable yang dicompile dengan Position Independent Executable
(PIE/PIC), lokasi section .text dan .data bisa saja berbeda dari awalnya; ketika
dimuat ke memori jika ASLR dari OS menyala.
:::
